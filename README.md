# Gym Workout Tracker

A simple mobile-friendly web app for generating and tracking gym workouts.

## How to use

1. Open `index.html` in your phone's browser
2. Pick a body area (Chest, Back, Legs, Arms, Shoulders)
3. A workout is randomly generated with 4 exercises and 3 sets each
4. Log your weight and reps as you go
5. Tap **Finish Workout** to see your summary
